# MCMV1---41M---Steel

Aplicação web **Avalia Imóveis** — Vite, React, mapas e viabilidade de empreendimentos.

Repositório: [github.com/rodrigofblopes/MCMV1---41M---Steel](https://github.com/rodrigofblopes/MCMV1---41M---Steel). Deploy estático via **GitHub Actions** → GitHub Pages (`npm run build:gh-pages`).

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deploy na [Vercel](https://vercel.com) (framework Vite, saída `dist`).
