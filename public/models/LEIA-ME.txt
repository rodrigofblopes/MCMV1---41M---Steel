Modelo 3D — aba Projeto > 3D

  O ficheiro usado pela app é:

  public/models/41M.glb

  Se exportar o SketchUp/Rhino para 41M.glb na RAIZ do projeto, copie para cá com:

    npm run sync:model

  Cada `npm run build` gera um novo parâmetro ?v=... no URL do GLB para o browser não manter
  uma versão antiga em cache (Vercel/GitHub).

Para mudar o nome, edite:
  src/pages/Apresentacao3D.jsx  (const MODELO_GLB)
